# Ferel Landing Page

Modern, high-conversion landing page for Ferel - AI-powered financial intelligence app.

## Tech Stack

- **Next.js 14** - React framework
- **Tailwind CSS** - Utility-first CSS
- **Framer Motion** - Smooth animations
- **Lucide React** - Beautiful icons

## Features

- Dark theme with glassmorphism effects
- Phone mockup in hero section
- Real-time market indices with sparklines
- Mobile-responsive design
- Smooth scroll animations

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Deploy

### Vercel (Recommended)

```bash
vercel --prod
```

Or deploy via Vercel dashboard: https://vercel.com/new

### GitHub

```bash
git init
git add .
git commit -m "Initial commit"
git push -u origin main
```
